# ngauth Branding Assets

This directory contains the official branding assets for the ngauth project.

## Logo Files

### Logo (512x512px)
- **File**: `logo.png` (PNG format, transparent background)
- **File**: `logo.svg` (SVG format, scalable vector)
- **Dimensions**: 512x512 pixels (PNG), scalable (SVG)
- **Usage**: GitHub repository, documentation, package managers, favicons

### Social Media Preview (1200x630px)
- **File**: `social-preview.png` (PNG format)
- **File**: `social-preview.svg` (SVG format, scalable vector)
- **Dimensions**: 1200x630 pixels (optimized for GitHub social preview)
- **Usage**: GitHub repository social preview, social media sharing

## Design Elements

The ngauth logo features:
- **Shield**: Represents security and protection
- **Key**: Symbolizes authentication and access control
- **Lock**: Emphasizes secure authorization
- **"ng" text**: Brand identifier at the bottom

### Color Palette

- **Primary Blue**: `#4299E1` (Shield fill)
- **Dark Blue**: `#2C5282` (Borders and key)
- **White**: `#FFFFFF` (Key and lock elements)
- **Dark Gray**: `#2D3748` (Text and backgrounds)
- **Light Gray**: `#A0AEC0` (Secondary text)

## Usage Guidelines

1. **Maintain aspect ratio**: Always preserve the original proportions
2. **Minimum size**: Don't scale below 64x64 pixels
3. **Clear space**: Provide adequate padding around the logo
4. **Background**: The logo works best on white or light backgrounds
5. **No modifications**: Don't alter colors, rotate, or distort the logo

## File Formats

- **PNG**: Use for web, documentation, and general purposes
- **SVG**: Use when scalability is needed (responsive designs, large prints)
